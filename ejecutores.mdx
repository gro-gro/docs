## Manual para Ejecutores

### 3.1 Antes de empezar

1. **Revisá la cola de issues `To‑Do`/`Bloqueados` de tu equipo cada mañana.**
2. Auto‑asignate lo que vas a tomar (atajo **I**). Si falta un issue, ¡créalo!

### 3.2 Crear un Issue rápido

1. **C** → *Template Rápida* (atajo **Alt + C**).
2. Completá:
   * Nombre siguiendo la nomenclatura.
   * Equipo.
   * Labels (CLIP, REEL, DISEÑO…).
3. Guardá (**Ctrl + Enter**).

### 3.3 Gestionar Status

* Al empezar → **S** → *En Edición*.
* Al terminar → **S** → *Para Revisar*.

### 3.4 Trackeo de horas

En la descripción agregar:

```
Horas: 8+8+3   ← bloques trabajados, sin espacios
```

> *Todos los issues deben tener horas cargadas.*

### 3.5 Shortcuts clave de Ejecutores

* **Crear issue:** `C`
* **Guardar:** `Ctrl + Enter`
* **Cambiar status:** `S`
* **Asignarme:** `I`
* **Establecer fecha límite:** `Shift + D`
* **Abrir menú comandos:** `Ctrl + K`
  (Ver [Referencia de Shortcuts](shortcuts) para la lista completa).

### 3.6 Buenas prácticas

* Todo lo que hagas debe tener su issue y horas.
* Comentá dentro del issue cualquier duda o material adicional.
* Usá la vista *Preview* (`Space`) para una revisión rápida.
